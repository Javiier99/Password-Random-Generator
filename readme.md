# 🔑 Generador de Contraseñas - Extensión para Google Chrome

¡Crea contraseñas seguras y aleatorias al instante! Esta extensión nació de la necesidad personal de contar con una herramienta rápida y eficaz para generar claves robustas directamente desde el navegador. 

Actualmente, la extensión ya se encuentra **publicada en la Chrome Web Store**.

---

## 🚀 Características actuales
* **Rapidez:** Genera contraseñas con un solo clic.
* **Seguridad:** Caracteres aleatorios que aumentan la robustez de tus cuentas además que no se guardan las contraseñas.
* **Accesibilidad:** Integrada directamente en tu barra de extensiones.

## 🔮 Próximas Mejoras (Roadmap)
El proyecto sigue en desarrollo. Próximamente añadiré nuevas funcionalidades para que sea aún más personalizable y útil para todos:
* 🔢 **Longitud personalizada:** Opción para elegir la cantidad exacta de caracteres que necesitas.
* 🔣 **Filtros avanzados:** Posibilidad de incluir o excluir caracteres especiales según los requisitos de cada sitio web.
